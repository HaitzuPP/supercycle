Robot hardware has fallen from $1M to under $100k in five years. The ownership infrastructure has not been built.

553kRobot installations (IFR 2023)
$675MFigure AI Series B
<$100kGeneral-purpose unit cost
"Without a shared map, every robot is working blind."
Auki Labs, Ep. 3
The Case

General-purpose robot units crossed below $100k this year. Five years ago the same capability cost over $1M. That 10x compression did not happen because of one innovation. Motors, sensors, and compute all deflated simultaneously. The pattern matches solar panels in 2010 and genome sequencing in 2012. Neither reversed.

When Amazon buys 100,000 robots, the economic return goes to Amazon equity. Decentralized ownership is a different architecture: productive robot infrastructure that individuals or collectives can own, deploy, and earn from directly. That model has no incumbent yet.

Robots operating in shared environments need a shared map. Without it, every robot is running blind. Building that spatial layer after mass deployment is the hard way. The window to set the standard is now, while the fleet is still small enough that early protocols can become the default.

The Case

200 million protein structures, resolved. That task used to take a PhD lab years per protein. AlphaFold2 did it in hours at scale. The bottleneck moved downstream: from knowing the structure to identifying candidate molecules worth synthesizing. That is exactly what competitive AI networks are now attacking.

No AI-discovered compound has cleared Phase III. That binary risk is real. It is also the opportunity: investors who wait for FDA confirmation will pay a premium over current entry prices. Isomorphic Labs raised $2B on the bet that it happens within this decade.

NumerAI's tournament model consistently outperforms closed quant funds on the same data. Drug candidate identification is the same type of problem: distributed signal beats concentrated effort when the problem is fundamentally a search across a large possibility space.

AI agents generate the majority of internet traffic. The infrastructure was built for browsers.

50%+Non-human internet traffic (Cloudflare)
$2B+Agent infrastructure funding (2024)
"The protocols that survive will be those that assume agents are the primary user, not a special case."
0xSammy, Ep. 1
The Case

Cloudflare's network data shows bot and agent traffic constitutes the majority of HTTP requests across the internet. Most of that traffic is hitting infrastructure built for browsers and human interaction. The friction compounds at every layer where the mismatch exists: authentication, payment, rate limiting, structured data return. Each friction point is a commercial opportunity.

When a human visits a bank website, HTTPS verifies the server. When an AI agent calls a bank API, there is no standard for verifying which agent is calling, on whose behalf, with what authorizations. The team that builds that standard sets the default for a decade.

Stripe captured mobile payments because they shipped a mobile-native API before the banks realized mobile was the primary channel. The window exists because incumbents are slow to re-architect for new primary users. Microsoft, Google, and Apple are building agent infrastructure now. The window closes when one of them ships something good enough to become the default.

Most DePIN has no structural advantage over centralized alternatives. Early protocol infrastructure is a different bet.

$5.9BMulticoin AUM
$0.04Early Solana seed price
12-18moDeveloper activity lead time
"Where developers build, users follow. By 12 to 18 months."
Shayon Sengupta, Multicoin Capital, Ep. 4
The Case

Solana passed the test in 2018 because 50,000 transactions per second was a real, measurable advantage over Ethereum's 15. Most DePIN networks fail that test: they are decentralized for its own sake, with token economics that do not survive the arrival of a well-funded centralized competitor.

GitHub commits and hackathon participation consistently preceded Solana's price move by over a year. Developers build where the technical architecture is compelling, regardless of token price. Tracking developer concentration gives you 12-18 months of lead time before the market prices it.

Bittensor and comparable AI infrastructure networks have growing developer activity, rising validator counts, and increasing subnet registrations: the same leading metrics that preceded Solana's price discovery. Institutions with meaningful AI infrastructure positions are not yet visible. The entry window that Multicoin had in 2018 at $0.04 is the analogue. It closes with the first major institutional announcement.
